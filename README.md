### Project desc

How to use mqtt in golang

### Getting Started
1. `$ git clone git@github.com:niomwungeri-fabrice/hello-go-mqtt.git` // clone the project
2. `go mod tiy` // download dependency
2. `$ go run main.go` // run the project
